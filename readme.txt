Please find the below steps to execute the test project

Method 1:-
1. Clone the project directly to your drive
2. navigate to the inside the project dir from terminal or from CMD(in case of windows), inside Automation dir
3. run following command -
mvn test
4. Test report will be generated in the project dir itself having name TestReport.html

Method 2:-
1. Import the project in eclipse as a Maven project
2. Right click on project and run as a maven test
3. Test report will be generated in the project dir itself having name TestReport.html

Method 3 :-

1. Import the project in eclipse as a Maven project
2. open testng.xml file and run testng suite.
3. Test report will be generated in the project dir itself having name TestReport.html
